Installing Noc

If you're using Linux, macOS (or BSD) (For the Windows users, use WSL)

Get the latest version

git clone https://github.com/mortim/noc && cd noc

Get the Noc v0.1.0.0

Download from this release.

Install Noc (for all versions)

make install

Building locally the Noc documentation (only the v0.1.0.0)

There is a tool written in Noc to generate the documentation (nocdoc), check the README of the 'docs' repos to build the Noc documentation.

You can also document your own functions, check the README of this tool.

When you have generated the documentation, you can check the starting page in "noc_documentation/index.html".

Updating and Uninstalling

We can reinstall Noc with the latest version.

git clone https://github.com/mortim/noc && cd noc

# or if have already the noc repository
git pull origin master

then

make update

To uninstall Noc.

make uninstall

Troubleshooting

To check if we have Noc installed:

noc --version
